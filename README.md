# Uldana Ospanbek — Portfolio Site

Двуязычный (RU/KZ) премиальный сайт-портфолио: AI marketing & sales architect.

## Структура

- `index.html` — весь сайт (разметка, стили, скрипты) одним файлом.
- `assets/img/` — фотографии для hero и блока «Обо мне».
- `serve.mjs` — локальный dev-сервер (`node serve.mjs` → http://localhost:3000).
- `screenshot.mjs` — Puppeteer-скриншоты страницы для проверки вёрстки.

## Запуск локально

```bash
npm install
node serve.mjs
```

Открыть http://localhost:3000

## Разделы сайта

Hero, Обо мне (с разворачиваемой полной биографией), Услуги, Портфолио (карусель), Пакеты, Аудитория, Отзывы (текст/видео/аудио карусели), Кейсы, Контакты, AI-ассистент (сценарийный чат), плавающая кнопка WhatsApp.
